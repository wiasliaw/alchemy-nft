# hardhat-boilerplate

This is a boilerplate for solidity development.

## Feature

- [X] linting & formatting
- [ ] testing
- [ ] coverage

## hardhat plugins

- hardhat-ethers
- hardhat-waffle
- hardhat-etherscan
- hardhat-typechain
- hardhat-gas-reporter

## linting & formatting

- typescript
  - linter: typescript-eslint
  - format: prettier
- solidity
  - linter: solhint
  - format: preitter

## testing

- solidity
  - library: ethersjs & waffle
  - type support: typechain
